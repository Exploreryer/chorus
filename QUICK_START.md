# Chorus Extension - Quick Start Guide

## üì¶ Installation Steps

### 1. Extract Extension Files
Extract `chorus-extension-v1.0.0.zip` to your computer.

### 2. Open Chrome Extensions Page
Enter in Chrome browser address bar:
```
chrome://extensions/
```

### 3. Enable Developer Mode
Turn on the "Developer mode" switch in the top right corner of the page.

### 4. Load Extension
1. Click "Load unpacked"
2. Select the extracted `chorus-extension` folder
3. Click "Select Folder"

### 5. Pin Extension Icon (Optional)
Click the Chrome toolbar's extensions icon (puzzle icon), find Chorus and click the pin icon.

## üí° Usage Tips

### Tip 1: Log In First
Before using the extension, it's recommended to log in to all AI products to be evaluated, so you can see results directly after distribution.

### Tip 2: Customize Product List
Click the "Manage" button to add, edit, or delete AI products, creating your exclusive evaluation list.

### Tip 3: Batch Testing
Prepare multiple Prompts, use the "Clear" button to quickly switch, and perform multiple rounds of comparison testing.

### Tip 4: Configure Selector
If a product's input box is not auto-filled, you can configure a custom selector in the management interface (right-click input box ‚Üí Inspect ‚Üí Copy selector).
